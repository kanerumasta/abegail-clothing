# Abegail Collective Figma Import

This folder contains a Figma-importable SVG mockup of the current landing page:

- `abegail-collective-landing.figma.svg`

## How to use

1. Open Figma.
2. Drag `abegail-collective-landing.figma.svg` into a Figma file.
3. If image links do not appear after import, drag the images from `public/images` onto the matching placeholders.

## Notes

Figma's native `.fig` format is proprietary, so this repo provides an editable SVG handoff instead. The artboard uses the same visual direction as the app: warm cream background, black featured panel, soft product cards, Playfair-style headings, Inter-style UI text, and the existing clothing images.
